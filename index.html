About the RGBA tool you see before you!

This tool was designed out of curiosity in order to learn JavaScript, Created during college around Christmas & New Year holiday of 2018!


Usage

If you see a colour you'd like to use for any purpose, highlight it, copy it and paste the rgba() value into your own code!
Not seeing an RGBA colour you like, simply refresh the page and wait for more rainbow!
